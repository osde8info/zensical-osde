Research In Motion (RIM) have just announced a [BlackBerry® Tablet OS](http://na.blackberry.com/eng/developers/tablet.jsp "blackberry") based on the QNX operating system for the [BlackBerry PlayBook](http://na.blackberry.com/eng/ "blackberry").

You can now [register for early access to the BlackBerry Tablet OS Software Developer Kit](https://www.blackberry.com/profile/?eventId=5005&choice=tablet "blackberry") (SDK).
