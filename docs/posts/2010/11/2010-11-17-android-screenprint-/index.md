summary

1. install java sdk
2. install android sdk to C:\\ProgFiles\\android-sdk-windows
3. run C:\\ProgFiles\\android-sdk-windows\\sdk manager.exe
4. choose usb drivers
5. goto App settings | Development | enable USB debugging
6. plug in android usb cable
7. install android usb driver from C:\\ProgFiles\\android-sdk-windows\\usb\_driver
8. run C:\\ProgFiles\\android-sdk-windows\\tools\\ddms.bat
9. wait 15 seconds
10. press CTRL-S (on your PC)
11. save screendump
