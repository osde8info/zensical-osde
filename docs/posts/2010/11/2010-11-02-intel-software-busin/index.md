Intel Software Network Communities are sections of the [Intel Software Network](http://software.intel.com/en-us/articles/intel-software-network-communities/ "intel") resourced _with Community Managers to support and sustain an ongoing and more responsive 2 way communication around key developer focus areas. These communities encourage knowledge sharing between developers and Intel through blog posts, forum discussions, and technical articles._
