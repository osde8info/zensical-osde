If you are getting the error **Error: This domain name has already been used as an alias or domain** when you are trying to move or swap a [Google Apps](http://www.google.com/apps/intl/en/business/index.html) domain alias or primary domain try either [waiting for at least five days](http://www.google.com/support/a/bin/answer.py?hl=en&answer=56238) or [delete the existing primary account in order to create a new primary account](http://www.google.com/support/a/bin/answer.py?hl=en&answer=56198).
