[IBM DevWorks Android Internet Data Tutorial](https://www.ibm.com/developerworks/xml/library/x-dataAndroid/) 

  

> _Many of your Android applications will need to interact with Internet data, which comes in a variety of formats._ 
> 
> _In this article, build an Android application that works with two popular data formats—XML and JavaScript Object Notation (JSON)—as well as the more exotic protocol buffers format from Google._ 
> 
> _And you'll learn about the performance and coding trade-offs associated with each format._
