first the good news ! ed/l/x/[ubuntu](http://www.ubuntu.com/) 10.4 will be a long term support (lts) release and with the help of [vmware player](http://www.vmware.com/products/player/) 3.0.1 (P4 or higher reqd) you can create a virtual machine to help [test the 10.4 beta](http://www.ubuntu.com/testing/lucid/beta1) today using vmwares zero auto install feature which also auto installs vmware tools & vmware toolbox !
