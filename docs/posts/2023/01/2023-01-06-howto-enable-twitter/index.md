howto enable @twitter #alt #image #accessibility mode #smn #bmn

if you post on @twitter be sure to enable the @twitter #alt #image #accessibility mode

you can find it on web and mobile apps under

- Accessibility | Media | Receive image description reminder = YES

```
Media
Receive image description reminder
Enables a reminder to add image descriptions before a Tweet can be sent.
```
