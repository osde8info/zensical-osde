the [motorola motodev](http://developer.motorola.com/docstools/motodevstudio/ "motorola motodev") eclipse ide for android allows you to run the android emulator inside an eclipse tab which you can drag outside the eclipse window if you prefer

i've also just discovered that it gives you a couple of extra buttons that let you ZOOM in and out and SWITCH LAYOUT which swaps between portrait or landscape screen orientations !
