## Ubuntu Name Changes

Ubuntu Netbook Edition and Ubuntu Desktop Edition _[have been combined into a single Edition](http://www.ubuntu.com/testing/natty/beta "ubuntu") called simply "Ubuntu"_

_(with no "Edition")._
