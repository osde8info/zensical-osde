[alfresco community](http://www.alfresco.com/community/ "alfresco") wcm quick start using a client and a server instead of having everything installed locally

- [download alfresco community edition](http://wiki.alfresco.com/wiki/Download_Community_Edition "alfresco") (it includes wcmqs)
- install alfresco onto server
- edit client host file and add

- serveripaddr edit.myalfresco
- serveripaddr live.myalfresco

- edit server host file and add

- 127.0.0.1 edit.myalfresco
- 127.0.0.1 live.myalfresco

- navigate and login to  http://edit.myalfresco:8080/alfresco
- navigate and login to  http://edit.myalfresco:8080/share
- follow the [alfresco "](http://wiki.alfresco.com/wiki/Web_Quick_Start_Installation_and_Configuration#Configuring_and_Accessing_Multiple_WebQS_Sites "alfresco")[create the web quick start site"](http://wiki.alfresco.com/wiki/Web_Quick_Start_Installation_and_Configuration#Configuring_and_Accessing_Multiple_WebQS_Sites "alfresco") instructions (note this is done in the Sites folder rather than the Web Projects folder !)
- navigate and login to  http://edit.myalfresco:8080/wcmqs
- follow the [alfresco "publishing content"](http://www.alfresco.com/help/34/community/wcmqs/user/ "alfresco") instructions
- navigate and login to  http://live.myalfresco:8080/wcmqs

you can even use alfresco as a front end to drupal by using the [drupal cmis module](http://drupal.org/project/cmis "drupal")
