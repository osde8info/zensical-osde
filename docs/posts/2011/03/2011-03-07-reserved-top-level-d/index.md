[IETF](http://tools.ietf.org/html/rfc2606 "ietf") Reserved Top Level DNS Names

- .example
- .invalid
- .localhost
- .test

[IETF](http://tools.ietf.org/html/rfc2606 "ietf") Reserved Second Level DNS Names

- example.com
- example.net
- example.org
