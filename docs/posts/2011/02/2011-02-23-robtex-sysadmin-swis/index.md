## [robtex](http://www.robtex.com/ "robtex") sysadmin swiss army knife internet tool(s)

- [RBL](http://www.robtex.com/rbl/) checks multiple RBL's if a list is specified
- [DNS](http://www.robtex.com/dns/) checks detailed dns information for a domain or a hostname
- [IP-number](http://www.robtex.com/ip/) checks ip number information such as dns reverse and forwards
- [C-net](http://www.robtex.com/cnet/) checks an entire c-network
- [whois lookup](http://www.robtex.com/whois/) checks whois information for a domain
- [route](http://www.robtex.com/route/) checks a specific routed prefix ()
- [AS numbers](http://www.robtex.com/as/) checks information on an AS-number
- [BGP announcements](http://www.robtex.com/bgp/) checks prefixes origined from a specific AS-number
- [AS macros](http://www.robtex.com/asmacro/) checks who belongs to an AS-macro (example: [as-ams-ix-peers](http://www.robtex.com/asmacro/as-ams-ix-peers.html))
- [RFC documents](http://www.robtex.com/rfc/) Request For Comments ([rfc2822](http://www.robtex.com/rfc/rfc2822.html))
- [DMOZ](http://www.robtex.com/dmoz/)
