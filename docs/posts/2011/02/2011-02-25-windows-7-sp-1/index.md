If you can't find Windows 7 SP 1 listed in your Windows Updates check that [KB976902](http://support.microsoft.com/kb/976902 "ms") is installed. You may save downloading the extra 50 updates before installing Windows 7 SP 1 !

Finally dont forget to run Windows Update after you have installed Windows 7 SP 1 to pick up any extra Windows Updates.
