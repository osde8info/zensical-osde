[opendns domain tagging](http://www.opendns.com/community/domaintagging/ "opendns domain tagging") is a crowd powered solution thats let you tag the web to make it safer for everyone (who uses [opendns](http://www.opendns.com/ "opendns") and its DNS servers)

for example you can choose the tags for [distrowatch](http://domain.opendns.com/distrowatch.com "opendns distrowatch tags") such as news/media or software/technology !
