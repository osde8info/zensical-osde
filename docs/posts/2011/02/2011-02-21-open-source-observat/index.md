The [Open Source Observatory and Repository](http://www.osor.eu/ "osor") for European public administrations (OSOR) _is a platform for exchanging information, experiences and FLOSS-based code for use in public administrations_ and includes

- a pan-European federation of [Open Source Software repositories](http://forge.osor.eu/ "osor forge")
- The [European Union Public Licence](http://www.osor.eu/eupl "eupl")
