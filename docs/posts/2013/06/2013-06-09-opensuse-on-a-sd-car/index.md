#opensuse on a SD card or USB stick with a persistent file system

- download ISO [software.opensuse.org/123/en](http://software.opensuse.org/123/en)
- write it to SD card or USB stick with [https://wiki.ubuntu.com/Win32DiskImager](https://wiki.ubuntu.com/Win32DiskImager)
- create a persistent file system [opensuse.org/Live\_USB\_stick#Live\_USB\_stick\_with\_persistent\_file\_system](http://en.opensuse.org/Live_USB_stick#Live_USB_stick_with_persistent_file_system)
