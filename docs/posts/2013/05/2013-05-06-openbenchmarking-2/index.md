to run the [openbenchmarking.org](http://openbenchmarking.org/) phoronix test suite on windows 7 you'll need to install

- php 5 [php.net/downloads.php](http://php.net/downloads.php)
- gtk+ [gtk.org/download/win32.php](http://www.gtk.org/download/win32.php)
- php-gtk [gtk.php.net/download.php](http://gtk.php.net/download.php)
- phoronix test suite [phoronix-test-suite.com/?k=downloads](http://www.phoronix-test-suite.com/?k=downloads)
