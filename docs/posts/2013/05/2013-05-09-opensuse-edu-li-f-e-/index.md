[opensuse.org/openSUSE:Education-Li-f-e](https://en.opensuse.org/openSUSE:Education-Li-f-e)

_openSUSE Education team is proud to present openSUSE-Edu Li-f-e (Linux for Education) based on openSUSE 12.3. The image is a "hybrid" iso image, the same image can be used to burn a Live DVD or to create a Live USB stick._

_This release includes carefully selected software for students, educators as well as parents. The software selection encompasses everything required to make computers productive for either home or educational use without having to install anything additional._
