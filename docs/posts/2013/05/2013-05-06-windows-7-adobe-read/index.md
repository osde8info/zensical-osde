on windows 7 64bit  i cant install or update adobe reader 9 without getting

```
Error 1303. The installer has insufficient privileges to access this directory: C:\Program Files (x86)\Adobe\Reader 9.0\Resource\CMap
```

[![windoz](http://farm9.staticflickr.com/8414/8713372992_064fbd7957.jpg)](http://www.flickr.com/photos/osde-info/8713372992/ "windoz by osde8info, on Flickr")

 

workaround

- uninstall adobe reader 9 (if it hasnt uninstalled itself)
- download adobe reader 10
- install adobe reader 10 (as administrator)
