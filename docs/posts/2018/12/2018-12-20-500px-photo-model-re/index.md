if you want to use [500px](https://iso.500px.com/) to sell your photos check out their model release and property release info

model release

_For any recognizable people in the photo, we require a model release to license the image commercially. Please attach all relevant releases. If you'd like, you can download a blank model release to use as a template._

property release

_If the photo contains private property, logos, brands, artwork, etc, we require a property release to license the image commercially. If you'd like, you can download a blank property release to use as a template._
