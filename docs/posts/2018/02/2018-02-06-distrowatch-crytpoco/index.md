The CryptoCoin.cc website _was first published on 1 December 2014 (initially as CryptoCoin.cm). It is a sister project of [DistroWatch.com](http://distrowatch.com/) (a popular online resource for news and information about free and open-source computer operating systems), reusing much of that site's PHP/SQL code. It was conceived out of the same desire as DistroWatch.com was back in 2001 - to create a database of cryptocurrencies and organise them in a logical and searchable manner_
