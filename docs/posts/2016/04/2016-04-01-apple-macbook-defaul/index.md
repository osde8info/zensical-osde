apple macbook defaults write hacks including getting screenshots to save in a subdir

- http://osxdaily.com/2012/10/09/best-defaults-write-commands-mac-os-x/
- https://developer.apple.com/library/mac/documentation/Darwin/Reference/ManPages/man1/defaults.1.html
- http://www.defaults-write.com/

eg

```
defaults write com.apple.screencapture location ~/Pictures/Screenshots
```
