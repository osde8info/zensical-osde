How to upgrade a DELL Latitude D630 from Windows 7 to Windows10 without it crashing or hanging.

Pre Install

- Upgrade BIOS to A19
- RESET BIOS to defaults
- Uninstall DELL System Detect
- Uninstall any third party antivirus
- Close all messenger apps

Now when you click on Get Windows 10 it shouldnt hang

Post Install

- Do not tweak BIOS settings after Windows10 is installed it may break WiFi
- Remove US keyboard
- Download UK language pack
