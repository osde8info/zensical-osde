the safe (and only way for some old thinkpad users) to try windoz10 is to download the windows 10 ISO and run it in an oracle virtual box vm

- download and install virtualbox https://www.virtualbox.org/
- download and create a windows 10 iso dvd https://osde8info.wordpress.com/2015/07/30/howto-create-your-own-windows10-dvd-or-usb-install-media/
- start vm
- skip licence key screen
