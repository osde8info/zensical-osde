Lenovo Thinkpad X201i v Lenovo Lenovo G50-30

Lenovo Thinkpad X201i

- Round 20V power connector
- FN then CTRL
- BIOS access via blue ThinkVantage button above keyboard

Lenovo Lenovo G50-30

- Square 20V power connector
- CTRL then FN
- BIOS access via black Novo button on left hand side
