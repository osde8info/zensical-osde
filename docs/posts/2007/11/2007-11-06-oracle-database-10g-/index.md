_Oracle Database 10__g Express Edition (Oracle Database XE) is a free-of-charge version of the world's most capable relational database. Oracle Database XE is easy to install, easy to manage, and easy to develop with._

_With Oracle Database XE, you use an intuitive, browser-based interface, to:_

- _Administer the database_
- _Create tables, views, and other database objects_
- _Import, export, and view table data_
- _Run queries and SQL scripts_
- _Generate reports_
