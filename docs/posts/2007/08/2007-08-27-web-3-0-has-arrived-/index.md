Web 3.0 has arrived ! Ning lets you create your own facebook apps with NO CODING REQUIRED !

Goto Ning [http://www.ning.com/](http://www.ning.com/)

Enter your network name and click Get your own social network

Goto My Networks [http://www.ning.com/?view=apps&op=using](http://www.ning.com/?view=apps&op=using)

Goto manage your network http://<yournetworkname>.ning.com/main/admin/manage

Click http://<yournetworkname>.ning.com/main/facebook/setup
