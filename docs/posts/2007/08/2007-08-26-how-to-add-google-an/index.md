How to add Google Analytics JavaScript to Ning

<code>  
<xn:head>  
<title>google analytics test</title>  
<meta name="description" content="google analytics test">  
<meta name="keywords" content="google, analytics, test">  
<script src="http://www.google-analytics.com/urchin.js" type="text/javascript">  
</script>  
<script type="text/javascript">  
\_uacct = "UA-123456-7";  
urchinTracker();  
</script>  
</xn:head>  
<code>
