You can even use PHP PEAR php\_beautifier without Apache !

Just  
\# aptitude install php-pear  
\# pear install php\_beautifier  
\# pear install channel://pear.php.net/php\_beautifier-0.1.13

Then  
$ php\_beautifier myfile.php
