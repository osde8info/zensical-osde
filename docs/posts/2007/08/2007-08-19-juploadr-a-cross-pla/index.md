This is AMAZING and will change the way you upload to flickr forever !

[Uploadr](http://juploadr.org/) is an open source cross platform, cross site Photo uploader.

Currently it runs on Linux, OS X and AnotherOS and supports both Flickr and Zooomr.

It allows you to set all properties of a photo before you upload it. It also supports batch editing, so you can make short work of uploading a bunch of files.
