Not by default BUT because [MEMSET](http://www.memset.com/index.php) use CentOS and therefore yum you can just login as root to your [MEMSET](http://www.memset.com/index.php) VM and run:

$ yum install ruby

$ ruby --version

ruby 1.8.5 (2006-08-25) \[i386-linux\]

So far [MEMSET](http://www.memset.com/index.php) is turning out to be the most flexible VPS VM webhosting solution I've ever seen !
