[MindTouch Deki Wiki](http://wiki.mindtouch.com/Deki_wiki) is a free open source wiki and application platform for communities and enterprises. Deki Wiki is an easy to use and sophisticated wiki for authoring, aggregating, organizing, and sharing content. Deki Wiki is also a platform for creating collaborative applications, or adding wiki capabilities to existing applications.

You can start using [MindTouch Deki Wiki](http://wiki.mindtouch.com/Deki_wiki) today by

- using a free-of-charge hosted version @ [http://www.wik.is/](http://www.wik.is/)  
    
- download a VMware virtual machine software appliance from [MindTouch Deki Wiki](http://wiki.mindtouch.com/Deki_wiki)
- download the source from [http://www.mindtouch.com/s/source.php](http://www.mindtouch.com/s/source.php)
- access the C# & PHP project on [http://sourceforge.net/projects/dekiwiki/](http://sourceforge.net/projects/dekiwiki/)

Install it yourself prerequisites

- Apache 2.x
- MySQL 5.0.X - NOTE: Deki Wiki Hayes uses stored procedures and is therefore incompatible with MySQL 4.x
- PHP 5.X - NOTE: Deki Wiki Hayes uses the PHP5 object model and is therefore incompatible with PHP4
- Mono 1.2.X
