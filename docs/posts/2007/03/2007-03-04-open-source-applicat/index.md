[OSAF](http://www.osafoundation.org/) is a non-profit organization developing next-generation inter-personal information management software.  

![](http://www.osafoundation.org/images/OSAFLogo.gif)

Their current projects include a desktop PIM application code-named "[Chandler](http://chandler.osafoundation.org/)", and a server code-named ["Cosmo"](http://cosmo.osafoundation.org/).
