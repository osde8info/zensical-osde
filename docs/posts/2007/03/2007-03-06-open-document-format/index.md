**ISO/IEC 26300, _Open Document Format for Office Applications (OpenDocument) v1.0_**  
has been designed to be used as a default file format for office applications with no increase in file size or loss of data integrity.

It will allow users to save and exchange editable office documents such as text documents (including memos, reports, and books), [spreadsheets](http://en.wikipedia.org/wiki/Spreadsheet), [databases](http://en.wikipedia.org/wiki/Database), [charts](http://en.wikipedia.org/wiki/Chart), and presentations â€“ regardless of application or platform in which the files were created.
