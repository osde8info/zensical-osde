[Sun Web Developer Pack](http://developers.sun.com/web/swdp/) (SWDP) is a new integrated toolkit from [Sun Microsystems](http://sun.com/) that consists of a collection of Web 2.0 technologies that enable next generation Web application development.

The toolkit consists of binaries, tutorial, documentation, samples (including source) to build your Web 2.0 applications and deploy them on industry-grade containers.

It includes support for building rich user interface using Ajax technologies with [Project jMaki](http://ajax.dev.java.net/) & [Project Dynamic Faces](http://jsf-extensions.dev.java.net/), light-weight Web services with [Atom](http://rome.dev.java.net/) / [REST APIs](http://developers.sun.com/web/swdp/docs/OnePagerREST_APIs.html) / [WADL](http://wadl.dev.java.net/) and server side scripting with [Project Phobos](http://phobos.dev.java.net/).
