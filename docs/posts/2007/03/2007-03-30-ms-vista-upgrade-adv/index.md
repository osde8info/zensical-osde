[MS Vista Upgrade Advisor](http://www.microsoft.com/downloads/details.aspx?familyid=42B5AC83-C24F-4863-A389-3FFC194924F8&displaylang=en) is designed to help XP users identify whether their PCs are ready for an upgrade to Windows Vista, which edition of Vista meets their needs, and which features of Vista will be able to run on their PCs.  
  
If you're PC isn't Vista compatible you could try [Ubuntu](http://www.ubuntu.com/getubuntu/download) instead !
