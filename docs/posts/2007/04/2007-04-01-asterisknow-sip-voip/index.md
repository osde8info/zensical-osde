Digium Asterisk® in minutes.

The most popular open source IP PBX software, Asterisk, can now be easily configured with a graphical interface. [AsteriskNOW](http://www.asterisknow.org/) is an open source Software Appliance; a customized Linux distribution that includes Asterisk, the AsteriskGUI™, and all other software needed for an Asterisk system.
