If you are already using a Tesco VOIP SIP softphone or USB Internet Phone it only takes you 60 secs to set up a Tesco Internet Phone Adapter (IPA 1000)

1. Plug in your Tesco Internet Phone Adapter (IPA 1000), ethernet and existing analogue / DECT phone  
    
2. Login to [https://www.tescointernetphone.com/account/](https://www.tescointernetphone.com/account/)
3. Click [Register my Internet Phone Adapter to my account](https://www.tescointernetphone.com/account/?group_id=ipa&tab_id=ipa_landing_page&mode=view)
4. Enter Serial number
5. Click Confirm
6. Wait for flashing red light to turn green

Start making FREE Tesco- to-Tesco VOIP SIP calls !
