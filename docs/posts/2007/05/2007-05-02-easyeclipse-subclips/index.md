[EasyEclipse for LAMP](http://www.easyeclipse.org/site/distributions/lamp.html), [subclipse](http://subclipse.tigris.org/) & [subversion](http://subversion.tigris.org/) (SVN) just gets better and better !

Using the [EasyEclipse for LAMP](http://www.easyeclipse.org/site/distributions/lamp.html) SVN Repository view lets you run almost any subversion command for example you can create subversion directories and create subversion tags & branchs !

Using the [EasyEclipse for LAMP](http://www.easyeclipse.org/site/distributions/lamp.html) Tasks view scans all the (PHP) source files in your project for comments containing the keyword "TODO" and displays a list of files that you can double click on and edit directly !
