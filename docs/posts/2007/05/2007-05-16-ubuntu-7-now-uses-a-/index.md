Ubuntu 7 now uses [Medibuntu](http://medibuntu.sos-sts.com/) (**M**ultimedia, **E**ntertainment & **D**istractions **I**n U**buntu**) which is a repository of packages that cannot be included into the Ubuntu distribution for legal reasons (copyright, license, patent, etc).

Some of these packages include the [![[WWW]](https://help.ubuntu.com/htdocs/ubuntu/img/u-www.png) libdvdcss](http://www.videolan.org/developers/libdvdcss.html) package from [![[WWW]](https://help.ubuntu.com/htdocs/ubuntu/img/u-www.png) VideoLAN](http://www.videolan.org/) and the external binary codecs package (commonly known as w32codecs) used by [![[WWW]](https://help.ubuntu.com/htdocs/ubuntu/img/u-www.png) MPlayer](http://www.mplayerhq.hu/) and [![[WWW]](https://help.ubuntu.com/htdocs/ubuntu/img/u-www.png) xine](http://xinehq.de/).

  
[https://help.ubuntu.com/community/AcrobatHowTo](https://help.ubuntu.com/community/AcrobatHowTo)  
**acroread** is no longer available in the Feisty repositories due to licensing issues. It can be easily installed by adding the [Medibuntu](https://help.ubuntu.com/community/Medibuntu) repository and installing the **acroread** package for the standalone reader and the mozilla-**acroread** package for the browser plugin.
