A short history of tagging

Tagging 1.0  

- just tags !  
    

Tagging 2.0  

- geo tags ?  
    

Tagging 3.0  

- triple tags or machine tags !  
    

Yahoo! flickr & Yahoo! upcoming have now enabled triple tagging or machine tagging !

This allow your flickr photos to automatically appear on the upcoming event they were taken at if you use the correct triple tag or machine tag

For example if you want your [Brighton Festival](http://upcoming.yahoo.com/event/188114/) flickr photos to appear in the [upcoming event](http://upcoming.yahoo.com/event/188114/) just tag them with the tag `upcoming:event=188114`

Machine tags are sometimes called triple tags because they consist of three parts:  

- namespace i.e. upcoming
- predicate i.e. event
- value i.e. value
