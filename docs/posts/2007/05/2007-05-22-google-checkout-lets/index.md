Google Checkout !

  

  
  
  
  
  
  
  
  
  
  
  
  
  
  
  
If you use the open source Joomla VirtueMart for your online shop(s) & store(s) you can now start using Google Checkout from within the open source Joomla VirtueMart module.  
  
There are two payment modules available that can be integrated into VirtueMart without much effort:

- [Google Checkout Payments module for VirtueMart](http://www.powersellersunite.com/googlecheckout.php) (commercial) by PowerSellersUnite
- [Google Checkout Virtuemart Plugin](http://extensions.joomla.org/component/option,com_mtree/task,viewlink/link_id,2132/Itemid,35/) by [OjO Templates](http://www.ojotemplates.com/) (free but requires registration).
