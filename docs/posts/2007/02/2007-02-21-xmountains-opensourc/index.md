If you'd to like to use [xmountains](http://osde-info.vox.com/library/post/xmountains.html) and are using \*buntu then just run synaptic package manager and type "xmountains" and then click install.

If you're using AnotherOs then I'd recommend either installing CygWin/X or using the [VMware Player](http://www.vmware.com/products/player/) and then then unzipping the [(Ubuntu) Browser Appliance](http://www.vmware.com/vmtn/appliances/directory/browserapp.html) virtual machine !
