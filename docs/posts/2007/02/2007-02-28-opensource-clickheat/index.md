[ClickHeat](http://www.labsmedia.com/clickheat/) is a "visual heatmap of clicks on a HTML page, showing hot and cold click zones".

It uses "Javascript on the client to track clicks, PHP and GD on the server to log clicks and generate the heatmap".

[http://sourceforge.net/projects/clickheat/](http://sourceforge.net/projects/clickheat/)
