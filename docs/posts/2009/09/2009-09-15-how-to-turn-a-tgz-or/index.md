If your favorite free libre open source project is only available as a tgz or zip (ie [umlet](http://www.umlet.com/)) you can now easily turn it into a fully viewable & searchable code repository by using [github](https://github.com/) or [google code svn](http://code.google.com/) !  
You can then use [ohloh.net](https://www.ohloh.net/) to analyse the code for you !

Simply  

1. sign up for a [github](https://github.com/) or [google code svn](http://code.google.com/) account
2. CREATE a new repo (ie [http://github.com/osde8info/UMLet](http://github.com/osde8info/UMLet))  
    
3. download tgz or zip
4. untgz or unzip
5. ADD files to repo
6. COMMIT files to repo
7. (if using github you may also need to 'remote add' and 'push')  
    
8. add enlistment to ohloh (ie [git://github.com/osde8info/UMLet.git](http://github.com/osde8info/UMLet.git))

Sit back and enjoy the [ohloh.net](https://www.ohloh.net/) code analysis and stats !  
[https://www.ohloh.net/p/UMLet/analyses/latest](https://www.ohloh.net/p/UMLet/analyses/latest)
