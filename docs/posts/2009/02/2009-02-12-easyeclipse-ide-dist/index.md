[EasyEclipse](http://www.easyeclipse.org/site/distributions/index.html) distributions are prepackaged releases of Eclipse that provides a fully functional Integrated Development Environment (IDE) for a specific type of development such as Java, PHP, Python or Ruby.

NOTE : Do not install [EasyEclipse](http://www.easyeclipse.org/site/distributions/index.html) in a directory with a **space** in it since it will instantly **crash** !

_Each distribution contains everything you need to start developing code, without any additional downloads. Note that there are_ _two releases for some distributions:_

- _EasyEclipse **1.2.2** is the previous stable release, based on [Eclipse](http://www.eclipse.org/) 3.2.2. All EasyEclipse plugins are working on this release._
- _EasyEclipse **1.3.1** is a new release based on [Eclipse](http://www.eclipse.org/) 3.3.1. Not all distributions, and not all plugins, are available yet in that release._
