the .eu domain is now supporting ACE Internationalised Domain Name (IDN) names which means that most european characters can now appear in the domain name part of URIs and URLs !

Examples:

- [déjà.eu](http://xn--dj-kia8a.eu/) (or [xn--dj-kia8a.eu](http://xn--dj-kia8a.eu/))
- [vis-à-vis.eu](http://xn--vis--vis-2ya.eu/) (or [xn--vis--vis-2ya.eu](http://xn--vis--vis-2ya.eu/))  
    

  
Note : type the second (ACE encoded) version in brackets if your keyboard doesn't have a [à](http://xn--vis--vis-2ya.eu/)[](http://caf%c3%83%c2%a9.eu/) character !
