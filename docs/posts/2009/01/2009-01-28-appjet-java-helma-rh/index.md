Apparently [AppJet](http://appjet.com/myapps) writes javascript apps using Java [helma](http://dev.helma.org/wiki/Sites+using+Helma/) and serves javascript apps using Java [rhino](http://www.mozilla.org/rhino/) !

[http://dev.helma.org/wiki/Sites+using+Helma/](http://dev.helma.org/wiki/Sites+using+Helma/)  
_Another browser based soft-coding framework, offered as a hosting service. The AppJet site itself runs on Helma, providing the environment in which the apps are developed, while the resulting apps themselves are served directly from a custom developed Rhino virtualisation engine._
