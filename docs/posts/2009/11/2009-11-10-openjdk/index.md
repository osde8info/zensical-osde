[OpenJDK](http://openjdk.java.net/) is _the place to collaborate on an open-source implementation of the [Java Platform, Standard Edition](http://java.sun.com/javase/), and related projects.  
_  
There are two [OpenJDK](http://openjdk.java.net/) streams:  

- [JDK6](http://openjdk.java.net/projects/jdk6/)  
    
- [JDK7](http://openjdk.java.net/projects/jdk7/)  
    

CentOS and OEL include a java-1.6.0-openjdk rpm in their yum repos and you can find java-1.6.0-openjdk-plugin rpms for Fedora 9,10 & 11 at:  

- [http://www.icewalkers.com/rpm/java-1.6.0-openjdk-plugin/fedora-core-9/](http://www.icewalkers.com/rpm/java-1.6.0-openjdk-plugin/fedora-core-9/download/java-1.6.0-openjdk-plugin-25884.html)
- [http://www.rpmfind.net/linux/rpm2html/search.php?query=java-1.6.0-openjdk-plugin](http://www.rpmfind.net/linux/rpm2html/search.php?query=java-1.6.0-openjdk-plugin)

You can test your [OpenJDK](http://openjdk.java.net/) Java SE plugin is working by running the java jre jvm demo examples that can be found at:  

- [http://java.sun.com/products/plugin/1.5.0/demos/plugin/applets.html](http://java.sun.com/products/plugin/1.5.0/demos/plugin/applets.html)
