Oracle Sun Java have announced betas of Java SE 7 and Java EE 6 !  

- [Java SE 7](https://cds.sun.com/is-bin/INTERSHOP.enfinity/WFS/CDS-CDS_Developer-Site/en_US/-/USD/ViewProductDetail-Start?ProductRef=jre-7-ea-b66-oth-JPR@CDS-CDS_Developer)

Java SE7 will have :  

- comprehensive support for file I/O
- Sockets Direct Protocol (SDP) to support stream connections over InfiniBand fabric
