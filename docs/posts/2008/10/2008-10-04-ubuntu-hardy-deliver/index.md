The alpha release of Ubuntu Delivered by rPath is now available as a platform selection on rBuilder Online.

Create the appliance using the latest version of [Ubuntu Delivered by rPath](http://blogs.conary.com/index.php/rpathcom/2008/09/30/changes_to_ubuntu_delivered_by_rpath).

Goto the rBuilder Online's web interface:

1. Click "Create /Edit product version"
2. Select "Ubuntu Hardy Delivered by rPath" from the drop down menu
