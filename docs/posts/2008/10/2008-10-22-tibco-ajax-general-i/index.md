[TIBCO](http://gi.tibco.com/) General Interface JavaScript library  
_  
General Interface is an open source project focused on enabling rapid creation of reliable Ajax applications, components and portlets with the look and feel of desktop graphical user interface applications. General Interface is aligned with the industry movement towards HTTP data services. Accordingly General Interface is intended to communicate with XML, SOAP, JavaScript and other HTTP accessible services both at a simple level or as part of larger SOA implementation._
