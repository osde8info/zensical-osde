[Easy Eclipse for LAMP](http://www.easyeclipse.org/site/distributions/lamp.html) which is fully multi-platform compatible including GNU/Linux & Apple Mac OS X now has add-ons, extensions, plug-ins for :

- [DataBase](http://www.easyeclipse.org/site/plugins/quantum.html)  
    
- [Bugzilla, Trac, and JIRA](http://www.easyeclipse.org/site/plugins/eclipse-mylyn.html)
- [SubVersion](http://www.easyeclipse.org/site/plugins/subclipse.html)
- [Trac & Subversion](http://www.easyeclipse.org/site/plugins/subclipse-mylyn.html) integration  
    
- [Log Watching](http://www.easyeclipse.org/site/plugins/graysky-logwatcher.html)
- [HTML](http://www.easyeclipse.org/site/plugins/eclipsetidy.html)
- [JavaScript](http://www.easyeclipse.org/site/plugins/eclipse-wtp-web.html)
- XML & [XSLT](http://www.easyeclipse.org/site/plugins/eclipsexslt.html)
- [QA & Testing](http://www.easyeclipse.org/site/plugins/simpletest.html)  
    

and you can find even more Easy Eclipse plug-ins @

- [http://www.easyeclipse.org/site/plugins/index.html](http://www.easyeclipse.org/site/plugins/index.html)  
    

  
to make it one of the best free libre open source IDEs for LAMP web developers !
