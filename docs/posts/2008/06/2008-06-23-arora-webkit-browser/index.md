_  
Arora is a simple cross platform web browser. Currently Arora is a very basic browser whose feature list includes things like "History" and "Bookmarks". It does not have support for netscape plugins, so no flash support until Qt 4.5. But it is small, less than 10,000 lines of code, very fast, lean, mean and loads of fun to hack on._

[http://code.google.com/p/arora/wiki/Goals](http://code.google.com/p/arora/wiki/Goals)

_To make rapid progress possible, we try to keep the code relatively easy to understand, straightforward algorithms and data structures are used when possible. Consistent code styling, clear, maintainable code, and we continue to improve names and code structure to aid understanding._
