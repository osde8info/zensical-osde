[Vuze](http://sourceforge.net/projects/azureus/) is the new name for the Azureus open source P2P Java based bit-torrent download client with 100's of settings/options and dozen's of plug-ins !

You can learn more about [Vuze](http://sourceforge.net/projects/azureus/) at :  

- [http://www.azureuswiki.com/index.php/Initial\_Setup\_Guide](http://www.azureuswiki.com/index.php/Initial_Setup_Guide)  
    
- [http://www.azureuswiki.com/index.php/Main\_Page](http://www.azureuswiki.com/index.php/Main_Page)
- [http://www.azureuswiki.com/index.php/Azureus\_Options](http://www.azureuswiki.com/index.php/Azureus_Options)
- [http://azureus.sourceforge.net/plugin\_list.php](http://azureus.sourceforge.net/plugin_list.php)

  
In fact there are so many settings/options in [Vuze](http://sourceforge.net/projects/azureus/) that it even has a search feature to help you find a particular option ! For example you can search for all "update" settings/options !
