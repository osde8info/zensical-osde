According to WikiPedia it depends if you are the EEC or not !

[http://en.wikipedia.org/wiki/.gb](http://en.wikipedia.org/wiki/.gb)

> _**.gb** is a reserved [Internet](http://en.wikipedia.org/wiki/Internet "Internet") [country code](http://en.wikipedia.org/wiki/Country_code "Country code") [top-level domain](http://en.wikipedia.org/wiki/Top-level_domain "Top-level domain") ([ccTLD](http://en.wikipedia.org/wiki/CcTLD "CcTLD")) for the [United Kingdom](http://en.wikipedia.org/wiki/United_Kingdom "United Kingdom"). Introduced at the same time as the UK's other top-level domain ([.uk](http://en.wikipedia.org/wiki/.uk ".uk")), it was never widely used. It is no longer possible to register under this domain.  
> _  

[Imperfect implementations](http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2)

> _Starting in 1985, two-letter codes have been used in the [domain name system](http://en.wikipedia.org/wiki/Domain_name_system "Domain name system") as [country code top-level domains](http://en.wikipedia.org/wiki/Country_code_top-level_domain "Country code top-level domain")[Internet Assigned Numbers Authority](http://en.wikipedia.org/wiki/Internet_Assigned_Numbers_Authority "Internet Assigned Numbers Authority") assign the ccTLDs, mostly following the ISO 3166-1 alpha-2 codes, but with a few exceptions (e.g., the [United Kingdom](http://en.wikipedia.org/wiki/United_Kingdom "United Kingdom") requested an [exceptional reservation](http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2#Exceptional_reservations) of [.uk](http://en.wikipedia.org/wiki/.uk ".uk") for their ccTLD instead of their native [.gb](http://en.wikipedia.org/wiki/.gb ".gb")).[\[1\]](http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2#cite_note-0)_ (ccTLDs). Currently, the
> 
> _The [WIPO](http://en.wikipedia.org/wiki/World_Intellectual_Property_Organization "World Intellectual Property Organization") coding standard ST.3 uses alpha-2 codes for encoding states, other entities and intergovernmental organizations (and their [Patent](http://en.wikipedia.org/wiki/Patent "Patent")/[Trademark](http://en.wikipedia.org/wiki/Trademark "Trademark") Offices), but includes a number of additional codes. For these additional codes, see the section_ _[Codes currently agreed not to use](http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2#Codes_currently_agreed_not_to_use)._
> 
> _[NATO country codes](http://en.wikipedia.org/wiki/NATO_country_code "NATO country code") are also based on ISO 3166-1 alpha-2._
> 
> _The [European Commission](http://en.wikipedia.org/wiki/European_Commission "European Commission") uses ISO 3166-1 alpha-2 with two exceptions: EL (not GR) is used for [Greece](http://en.wikipedia.org/wiki/Greece "Greece"), and UK (not GB) is used for the [United Kingdom](http://en.wikipedia.org/wiki/United_Kingdom "United Kingdom").[\[2\]](http://en.wikipedia.org/wiki/ISO_3166-1_alpha-2#cite_note-1)_
> 
> _The [United Nations](http://en.wikipedia.org/wiki/United_Nations "United Nations") uses a combination of ISO 3166 two- and three- letter codes, along with codes that pre-date ISO 3166, for [international vehicle registration codes](http://en.wikipedia.org/wiki/List_of_international_vehicle_registration_codes "List of international vehicle registration codes"), the codes on the oval stickers used to identify foreign road vehicles._
