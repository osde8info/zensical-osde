There's an rPath GNU/Linux distro / software appliance / virtual machine for EVERYONE !

Here are some very specific examples :

- [Concourse CRM](http://www.rpath.org/rbuilder/project/concourse/)  
    
- [SandBox](http://www.rpath.org/rbuilder/project/rpath-sandbox/)
- [Small Talk](http://www.rpath.org/rbuilder/project/syx/)  
    
- [USB](http://www.rpath.org/rbuilder/project/vestlinkux/)

and if you can't find one you like you can [rBuild](http://www.rpath.org/rbuilder/) your own !
