Thank you [tulrich](http://www.tulrich.com/index.html) for blogging (and creating ?) the [Google Earth FireFox Plug-in](http://google-latlong.blogspot.com/2008/05/google-earth-meet-browser.html) and the Google Earth [Monster Milk Float](http://www.google.com/earth/plugin/examples/milktruck/) Game !

_You can write [Google Earth Plug-in apps](http://google-latlong.blogspot.com/2008/05/google-earth-meet-browser.html) using Javascript, including game-like things. It's pretty fun. It also integrates smoothly with Google Maps API sites; basically you just enable 3D, and most things will magically work._

[http://www.google.com/earth/plugin/examples/milktruck/](http://www.google.com/earth/plugin/examples/milktruck/)
