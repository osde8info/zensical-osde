[GoogleMail](http://mail.google.com) now works with most POP and IMAP mail clients

**POP** ([http://mail.google.com/support/bin/answer.py?answer=13287)](http://mail.google.com/support/bin/answer.py?answer=13287)  

- pop.googlemail.com:995 SSL  
    
- smtp.googlemail.com:465 or 587 START TLS  
    
- Google Mail username (including @googlemail.com)
- Google Mail password

  
**IMAP** ([http://mail.google.com/support/bin/answer.py?answer=78799)](http://mail.google.com/support/bin/answer.py?answer=78799)  

- imap.googlemail.com:993 SSL  
    
- smtp.googlemail.com:465 or 587 START TLS  
    
- Google Mail username (including @googlemail.com)
- Google Mail password
