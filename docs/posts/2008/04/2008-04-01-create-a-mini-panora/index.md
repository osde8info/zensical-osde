Upload your photos to [Panoramio](http://www.panoramio.com) and geotag them !

Then add the following into an iframe on your site  

[http://www.panoramio.com/plugin.php?lt=51.5&ln=-0.1&z=5](http://www.panoramio.com/plugin.php?lt=51.5&ln=-0.1&z=5 "blocked::http://www.panoramio.com/plugin.php?lt=51.5&ln=-0.1&z=5")

Set these parameters  

- lt = latitude
- ln = longitude
- z = zoom (1-10)

and optionally add these parameters  
  

- k = kind of map, 0 = map, 1 = satellite, 2 = hybrid
- user = youruserid !
