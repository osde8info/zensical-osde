[Oracle JDeveloper](http://www.oracle.com/technology/products/jdev/index.html) is not just for Java !

You can also use it for [XML development](http://www.oracle.com/technology/products/jdev/viewlets/1013/xml_viewlet_swf.html) and it also has the following UML modeling tools:

- Class Modeler
- Use Case Modeler
- Activity Modeler
- Sequence Modeler
