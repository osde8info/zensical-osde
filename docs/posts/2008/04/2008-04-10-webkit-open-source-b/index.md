[WebKit](http://webkit.org/projects/goals.html) is an open source web content engine for browsers and other applications.

[WebKit](http://webkit.org/projects/goals.html) _value real-world web compatibility, standards compliance, stability, performance, security, portability, usability, and relative ease of understanding and modifying the code (hackability)._
