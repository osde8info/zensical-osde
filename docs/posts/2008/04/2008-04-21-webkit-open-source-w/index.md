_[WebKit](http://webkit.org/) is an open source web browser engine. WebKit is also the name of the Apple Mac OS X system framework version of the engine that's used by [Safari](http://www.apple.com/safari/), Dashboard, Mail, and many other OS X applications._

_[](http://webkit.org/)_WebKit is also used in:

- Adobe AIR
- [S60WebKit](http://trac.webkit.org/projects/webkit/wiki/S60Webkit) is used as the rendering engine behind the [Web Browser for S60](http://www.s60.com/browser).
