I've finally twigged how to use [MyBlogLog](http://www.mybloglog.com/buzz/members/osde-info/) ! Yahoo have done it again by buying the best Web 2.0 & Web 3.0 social media network aggregator I've seen yet !

1. Join [MyBlogBlog](http://www.mybloglog.com/buzz/members/osde-info/)
2. Add your blogs / sites / invisible wikki's etc, etc  
    
3. Add the [MyBlogBlog](http://www.mybloglog.com/buzz/members/osde-info/) JavaScript to your blogs / sites
4. Authenticate your sites (by adding (& deleting !) a post or META tag)  
    
5. Add your [Social Media Networks](http://social-media-networks.groups.vox.com/) and [Business Media Networks](http://business-media-networks.groups.vox.com/) (they integrate with almost all of them such as del.icio.us, upcoming, last.fm,  flickr, youtube, pownce,  jaiku, etc, etc)  
    
6. Create a network of like minded social media networkers just like you would do on del.icio.us, flickr, last.fm or VOX  
    
7. **Enjoy a single portal** to all your co - social media networkers blogs, posts, pownces, twits, etc, etc !  
    
8. Finally send a feature request to [MyBlogBlog](http://www.mybloglog.com/buzz/members/osde-info/) asking them for VOX integration / support !
