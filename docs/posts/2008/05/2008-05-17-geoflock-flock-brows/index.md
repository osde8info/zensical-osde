[**geoFlock**](http://flock.spatialviews.com/geoflock_features.html) gives you _a suite of in-built maps and mapping tools for your web browser (now based on [Minimap Firefox version](http://minimap.spatialviews.com/), but retaining geotag dialogs)._

_Create and save a sidebar map using the addresses or address links you find on web pages, or by manually adding locations. View traffic Info, drag and drop kml files, Geotag Flickr photos within the Flock photo uploader, geotag blog posts within the blog editor (including microformat geo class values, view in google maps/google earth links and insertion of quikmaps.com maps).
