three mobile data costs vary a lot !

Direct access on phone:  
   

- £3/M = **£3,000/GB** in Europe
- £2/M = £2,000/GB in UK
- £0.25/M (pre-pay 10M/m) = £250
- £5/G (pre-pay 1G/m) = £5

PC access via phone modem :

- £10/G (pre-pay 1G/m) = £10
- £5/G (pre-pay 3G/m) = £5
