Build your own SteamOS gaming PC for Steam games

- https://store.steampowered.com/steamos
- https://store.steampowered.com/steamos/buildyourown
- https://store.steampowered.com/steamos/oem
- https://steamcommunity.com/groups/steamuniverse/discussions/1/648814395741989999/

You will need a PC with

- Processor: Intel or AMD 64-bit capable processor
- Memory: 4GB or more RAM
- Hard Drive: 200GB or larger disk
- Video Chipset: AMD (RADEON 8500 and later) or Intel Graphics or NVIDIA
- UEFI Firmware (recommended)

SteamOS DOES NOT run in a oracle virtual box VM and although SteamOS is based on free libre open source software (debian) SteamOS is NOT free libre open source software.
