you may like to try and enable the ms #windows 10 windows security core isolation memory integrity security feature that _uses hardware virtualization and Hyper-V to protect Windows kernel mode processes from the injection and execution of malicious or unverified code._

- [Memory integrity - Windows security | Microsoft Docs](https://docs.microsoft.com/en-us/windows/security/threat-protection/device-guard/memory-integrity)
- [Device protection in Windows Security (microsoft.com)](https://support.microsoft.com/en-us/windows/device-protection-in-windows-security-afa11526-de57-b1c5-599f-3a4c6a61c5e2)

however you may not be able to enable it if you have certain Ricoh, SunPlus or ZTE hardware drivers installed
