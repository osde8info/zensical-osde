windoz env environment vars

set an env var with windoz cmd  
`set CLOUDINARY_URL=cloudinary://123456:abcdefghijkl@cloud_name`

set an env var with windoz ps  
`$Env:CLOUDINARY_URL="cloudinary://123456:abcdefghijkl@cloud_name"`
